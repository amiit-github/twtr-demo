# Mission Itch

### ===== Setup Environment on Mac =======

Install Node

`brew install node`

`brew update`

`brew upgrade node`


### ================ Install Muffin ================

The frontend stack uses Backbone.js and CoffeeScript, and two backend stacks include Google App Engine and Node.js/MongoDB.


`sudo npm install -g muffin.io`

### ================ Install mongoDB ================

`brew update`

`brew install mongodb`

### ============== Start server ================
`cd <project-name>`

`muffin watch -s`
